## 🚀 Projects

- **[Motion Planning Algorithms Benchmark and Comparison](https://github.com/rabiken/ned2-motion-planning-benchmark)** – My bachelor's thesis project where I benchmarked and compared various motion planning algorithms for a real robotic arm Niryo Ned2. 
  *Tech stack:* [ROS, C++, Python, Motion Planning Algorithm, etc.]

- **[ML-Based Bearing Fault Detection](https://github.com/rabiken/bearing-fault-detection-wst)** – Bearing Fault Detection using Wavelet Scattering Transform and Machine Leawrning. This project involved signal processing and machine learning techniques to detect faults in bearings.
  *Tech stack:* [Python, SVM, WST, Signal Processing, etc.]  

- **[Path Finding Algorithm Visualization](https://github.com/rabiken/PFA_visualization)** – A project where I implemented and visualized various pathfinding algorithms (A*, Dijkstra, BFS, DFS, etc.) in a grid environment.

  *Tech stack:* [C++, A*, Dijkstra, BFS, DFS, etc.]

- **[Maze Generator](https://github.com/rabiken/maze_generator)** – A toy project where I implemented a maze generator using the DFS algorithm. This project was a fun way to practice algorithm implementation and visualization.
  *Tech stack:* [DFS, Pygame etc.]
